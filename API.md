# API Endpoints

## Core URLs

- Gateway: [http://localhost:8080](http://localhost:8080)
- Frontend: [http://localhost:4200](http://localhost:4200)

## Auth

- `POST /auth/register`
- `POST /auth/login`
- `POST /auth/refresh`

## Catalog

- `GET /api/products`
- `GET /api/products/search?q=...`
- `GET /api/categories`
- `GET /api/brands`
